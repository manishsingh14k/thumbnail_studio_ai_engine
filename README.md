# thumbnail_studio_ai_engine
Vision-transformer and multimodal ML platform that predicts video thumbnail CTR, generates visual saliency heatmaps, and provides LLM-driven actionable design recommendations using title-image alignment.


## 🛠️ Key Features

*   **Multimodal CTR Prediction:** Evaluates visual features conditioned against video titles, category vectors, and localized competitor benchmarks.
*   **Visual Saliency Mapping:** Generates 2D spatial gaze-trajectory heatmaps using fine-tuned Vision Transformers (ViT) to pinpoint attention drop-offs.
*   **UI Safe-Zone Analysis:** Programmatically detects collision vectors between thumbnail text/subjects and platform metadata overlays (e.g., timestamps).
*   **Actionable Recommendation Engine:** Translates raw model embeddings into localized, textual design modifications (e.g., contrast adjustment, scaling metrics).
